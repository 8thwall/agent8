# 8th Wall Agent Modes

Choose a mode depending on your task:

- **Agent** – Write code, modify your scene, fix bugs, and answer questions
- **Code Only** – Create and edit code files only (no scene changes)
- **Scene Only** – Create and modify scene entities (no code changes)
- **Ask** – Ask questions about your codebase or 8th Wall (no changes made)
