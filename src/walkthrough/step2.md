# Create a new project

Select **➕ New Project** to create a new empty project.

<img src="images/new-project.gif" alt="8th Wall login" />

# Or, start from a template

To start from a template, Select a **AR Starters** or  **All Templates** to select one to clone.

Select a project thumbnail to open an existing project in your Workspace.

<img src="images/open-externally.png" alt="8th Wall login" />

Once your project is open in 8th Wall Desktop, click **Open Externally** in the left side Files panel to open the same project in VS Code.
